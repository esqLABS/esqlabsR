# Get the path to example ProjectConfiguration.xlsx

Get the path to example ProjectConfiguration.xlsx

## Usage

``` r
exampleProjectConfigurationPath()
```

## Value

A string representing the path to the example ProjectConfiguration.xlsx
file

## Examples

``` r
exampleProjectConfigurationPath()
#> [1] "/home/runner/work/_temp/Library/esqlabsR/extdata/examples/TestProject/ProjectConfiguration.xlsx"
```
