# Create a plotConfiguration or exportConfiguration objects from a row of sheet 'plotConfiguration' or 'exportConfiguration'

Create a plotConfiguration or exportConfiguration objects from a row of
sheet 'plotConfiguration' or 'exportConfiguration'

## Usage

``` r
.createConfigurationFromRow(defaultConfiguration, ...)
```

## Arguments

- defaultConfiguration:

  default plotConfiguration or exportConfiguration

- ...:

  row with configuration properties

## Value

A customized plot- or exportConfiguration object
