# Validate and process the 'exportConfiguration' sheet

Validate and process the 'exportConfiguration' sheet

## Usage

``` r
.validateExportConfigurationsFromExcel(dfExportConfigurations, plotGrids)
```

## Arguments

- dfExportConfigurations:

  Data frame created by reading the 'exportConfiguration' sheet

- plotGrids:

  List of multipanel plots created previously

## Value

Processed `dfExportConfigurations`
