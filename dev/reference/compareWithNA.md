# Compare values including `NA`

Compare values including `NA`

## Usage

``` r
compareWithNA(v1, v2)
```

## Arguments

- v1:

  Value or a list of values to compare. May include `NA`.

- v2:

  Value or a list of values to compare. May include `NA`.

## Value

`TRUE` wherever elements are the same, including `NA`'s,
