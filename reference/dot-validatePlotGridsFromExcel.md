# Validate and process the 'plotGrids' sheet

Validate and process the 'plotGrids' sheet

## Usage

``` r
.validatePlotGridsFromExcel(dfPlotGrids, plotIDs)
```

## Arguments

- dfPlotGrids:

  Data frame created by reading the ' plotGrids' sheet

- plotIDs:

  IDs of the plots that are referenced in the plot grids

## Value

Processed `dfPlotGrids`
