# Sample a random value from a distribution

Sample a random value from a distribution

## Usage

``` r
sampleRandomValue(distribution, mean, sd, n)
```

## Arguments

- distribution:

  The type of the distribution the random variable is to be sampled
  from. See `Distributions` for the list of supported entries.

- mean:

  Mean value of the random variable

- sd:

  Standard deviation of the random variable

- n:

  Size of the sample

## Value

Numerical vector of size n with randomly sampled values
