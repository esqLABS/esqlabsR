# Check if the `object` contains active binding with the name `field`

Check if the `object` contains active binding with the name `field`

## Usage

``` r
.validateClassHasField(object, field)
```

## Arguments

- object:

  A class or an instance of a class to check

- field:

  Name of the field

## Value

`TRUE` if the `object` has an active binding `field`, `FALSE` otherwise.
